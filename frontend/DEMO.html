RAG Chatbot - Translation Demo

Welcome to RAG Chatbot

This is a demonstration of the full-page translation feature. Click the translate button in the navbar above to see the entire page translate to Urdu.

Note: This is a visual demo showing the UI and button placement. In a real Docusaurus site, clicking the button will call the backend API to translate all content.
Status: Idle

Features

  • Translate entire pages with one click
  • Persistent translation across navigation
  • Preserves all layout and styling
  • Works with dynamic content
  • Graceful error handling

How It Works

  1. User clicks the translate button in the navbar
  2. System collects all text nodes from the page
  3. Sends batched request to translation API
  4. Updates DOM with translated content
  5. Stores original text for restoration

Technical Details

The translation system uses the following technologies:

  • TreeWalker API for efficient DOM traversal
  • React Context for state management
  • localStorage for persistence
  • Custom hooks for translation logic

Button Variants

The navbar button comes in three variants:

  • Default - Filled background (current)
  • Minimal - Transparent background
  • Outline - Border style

Integration Steps

To integrate this into your Docusaurus site:

  1. Wrap your app with PageTranslationWrapper
  2. Add NavbarTranslateButton to your navbar
  3. Configure in docusaurus.config.js

See INTEGRATION_EXAMPLE.md for detailed instructions.
